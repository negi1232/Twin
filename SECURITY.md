# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 1.5.x   | :white_check_mark: |
| < 1.5   | :x:                |

## Reporting a Vulnerability

セキュリティの脆弱性を発見した場合は、**公開 Issue を作成せず**、以下の方法で報告してください。

### GitHub Security Advisories (推奨)

1. [Security Advisories](https://github.com/negi1232/Twin/security/advisories/new) から非公開レポートを作成

### 報告に含めてほしい情報

- 脆弱性の概要
- 再現手順
- 影響範囲
- 可能であれば修正案

### 対応フロー

1. 報告受領後 **48 時間以内** に確認の返信
2. 脆弱性の評価と修正方針の決定
3. 修正パッチの作成とリリース
4. 報告者へのクレジット（希望する場合）
